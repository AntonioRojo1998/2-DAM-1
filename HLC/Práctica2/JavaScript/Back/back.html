Frameworks Back-end para JavaScript: Express.js

Express es un framework web para Node, que usa como lenguaje JavaScript. El objetivo de Express es ser minimalista, aún aportando funciones y herramientas robustas. El objetivo al usar Express es crear una API de forma rápida y sencilla. Algunos frameworks que usan Express son Sails, Poet, y Kites.

Veamos un ejemplo de una aplicación con Express:

const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => {
  res.send('¡Hola mundo!')
})

app.listen(port, () => {
  console.log(`Aplicación de ejemplo escuchando en el puerto ${port}`)
})

Esta aplicación va a iniciar un servidor y escuchar solicitudes de conexión en el puerto 3000. La aplicación responderá con "¡Hola mundo!" a las solicitudes que quieran acceder a la URL raíz (/), y responderá con el error 404 a las demás.

Ya vemos que para usar Express en nuestro entorno sólo hay que incluirlo en nuestro código JavaScript usando require('express')

. Previamente tendremos que tener instalados Node.js (instalable directamente desde su página) y npm ( Node Packet Manager).
