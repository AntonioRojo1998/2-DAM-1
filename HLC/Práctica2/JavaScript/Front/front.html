Frameworks Front-end para JavaScript: React.js

React es una librería de JavaScript para desarrollar interfaces de usuario, y está diseñado para que sea adoptado gradualmente, de forma que se pueda ajustar la cantidad de React necesaria en una plataforma.
Para usar React es necesario instalar Node.js. Cuando vayamos a crear una aplicación React, usaremos el siguiente comando: 

npx create-react-app <nombre-aplicacion>

Para añadir React a un archivo HTML sólo es necesario incluir un enlace al script .js que contiene el código.
Y con esto ya podríamos usar React en nuestros propios archivos JavaScript aplicados al archivo HTML.

Guías para React

React aporta dos formas de aprenderlo: una más práctica y otra más teórica. Aunque tal vez no te haga falta ir muy lejos, ya que la propia página principal de React ya tiene algunos ejemplos abiertos a modificaciones.
Una página que me ha ayudado a comprender cómo se usa React es Pensando en React. En esta página se habla de cuáles son los "caminos" que toma una mente que entiende React y tiene el objetivo de usarlo para crear una interfaz.

Un ejemplo con React

Ahora veremos un pequeño ejemplo de React: un juego de tres en raya. React permite componer IUs complejas a partir de pequeñas piezas llamadas componentes. Se crean clases que extienden dichos componentes, los cuales aceptan parámetros (llamados props) y devuelven una jerarquía de vistas a mostrar a través del método .

El método render devuelve una "descripción" de lo que tiene que mostrar en pantalla. React usa esta descripción para mostrar el resultado. Concretamente, el método render devuelve un elemento de React. Para indicarle a este método la descripción de lo que tiene que mostrar, se usa comúnmente una sintaxis llamada JSX, que facilita la escritura de las estructuras HTML. Veamos un ejemplo comparativo entre usar la estructura HTML y JSX para usar el método render: 

class ShoppingList extends React.Component {
  render() {
    return (
      <div className="shopping-list">
        <h1>Lista de compras para {this.props.name}</h1>
                <ul>
          <li>Instagram</li>
                <li>WhatsApp</li>
                <li>Oculus</li>
                </ul>
      </div>
    );
  }
}

// En vez de usar HTML, usando JSX sería:
/*#__PURE__*/
React.createElement("div", {className: "shopping-list"},
    React.createElement("h1", null, "Shopping List for ", props.name),
    React.createElement("ul", null,
    React.createElement("li", null, "Instagram"),
    React.createElement("li", null, "WhatsApp"),
    React.createElement("li", null, "Oculus")));

Como puedes apreciar, la sintaxis JSX tiene un estilo más acercado a la programación tradicional. Puedes usar cualquiera de las dos sintaxis.
