What is this website about?

The main content of this website is about CSS3.

Here, you will learn how to add styles to your own website.

What is CSS3?

CSS3 stands for Cascading Style Sheets.

It is a language we use to style an HTML document and describes how HTML elements should be displayed.

How CSS3 works?

To apply styles to HTML opages, a separated file is used, a style sheet with the .css extension.

When these two documents reach the browser, it will read the HTML document, apply the CSS styles to it, and display it.

Beginning

First of all, if you want to add styles to an HTML document, you have to create it beforehand.

Once you have done it, you can start creating an empty file where you are going to write the style of the website.

Before starting to write, we have to know that there are a lot of selectors that we can use in our CSS3 file, in order to save time.

Some of this selectors are: h1, h2, p, title, etc...

They are going to make our work easier, because we can edit all content of a single selector, even if there are more than 1,

and it will take a few seconds to do it.

Syntax

The systax is divided in 4 parts.

Selector: it is a pointer to the HTML element you want to style. Example: h1, p.

Declaration: it is a section in a CSS3 style sheet that declares the style you want to put in a selector. Example: {color:blue;}

Property: it is the way you want to style a selector. Example: color, text-align.

Value: it is the value you set to a property to style a selector. Example: red, center.

In this example, "aside" is the selector, the left parts starting from the ":" are the properties and the right parts are the values.

How to add CSS3 to an HTML file.

There are 3 ways to add CSS to an HTML file:

External:

With an external style sheet, you can change the look of an entire website by changing just one file!

Each HTML page must include a reference to the external style sheet file inside the link element, inside the head section.

Internal:

An internal style sheet may be used if one single HTML page has a unique style.

The internal style is defined inside the style element, inside the head section.

Inline:

An inline style may be used to apply a unique style for a single element.

To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

Some things you can change with CSS3.

  • Background.
  • Colors.
  • Borders.
  • Margins.
  • Padding.
  • Height/Width.
  • Text.
  • Font.
